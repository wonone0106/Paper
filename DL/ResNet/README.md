# Deep Residual Learning for Image Recognition

## 초록

모델은 기울기 소실 때문에 깊이가 깊을 수록 학습이 잘 되지 않는다. 

이걸 해결하기 위해 잔차 학습 프레임 워크를 제시한다!

잔차 네트워크를 도입하면 기울기 소실 문제를 해결하여 학습도 잘 되고 모델 깊이를 늘려 즉 표현력을 증가시켜 정확도 또한 증가된다.

이러한 방식을 이용해 대회에서 1등 했다.

## 서론
깊은 신경망일수록 표현력이 증가한다. 따라서 모델을 깊게 만들어야 정확도도 올라가며 최고의 성과를 보여준다.

근데 깊이만 늘리면 좋은 것은 아니다. 기울기 소실/폭주 등으로 깊이가 깊어지면 정확도가 급격히 떨어지는 현상이 일어난다.
